# SF-Hackaton3-FlickrApp
 Приложение для просмотра изображений от Flickr. В приложении есть два экрана — один для вида ленты фотографий, второй — для полноэкранного просмотра изображения.
 
 На первом экране должна быть реализована поисковая строка. Между экранами должна быть реализована полноценная навигация. Также на первом экране нужна кнопка для сброса результатов поиска. 
 
 Необходимые сторонние компоненты

  - Бесплатное API Flickr — зарегистрируйтесь здесь и получите свой ключ;
  - Ключ к API;
  - SDWebImage или любая другая библиотека для работы с изображениями;
  - Alamofire/Moya, либо стандартные средства;
  - SwiftyJSON для удобства работы с JSON.
